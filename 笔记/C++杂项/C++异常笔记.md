# C++ 异常（Exception）笔记

## 1. 什么是异常？

异常（Exception）用于处理**程序运行过程中发生的错误**。

例如：

``` cpp
v.at(100);
```

数组只有 5 个元素，却访问第 100 个元素，这就是运行时错误。

------------------------------------------------------------------------

## 2. throw：抛出异常

``` cpp
throw std::out_of_range("下标越界");
```

含义：

> 发现错误，主动抛出一个异常。

需要头文件：

``` cpp
#include <stdexcept>
```

------------------------------------------------------------------------

## 3. try-catch：捕获异常

``` cpp
try
{
    cout << v.at(100) << endl;
}
catch (const std::out_of_range& e)
{
    cout << e.what() << endl;
}
```

含义：

-   `try`：尝试执行代码。
-   `catch`：如果发生异常，就在这里处理。

------------------------------------------------------------------------

## 4. 不写 try-catch 会怎样？

``` cpp
cout << v.at(100) << endl;
```

如果 `at()` 中执行了：

``` cpp
throw std::out_of_range("下标越界");
```

程序会直接终止。

因此：

-   `throw`：负责抛出异常。
-   `try-catch`：负责捕获异常。

------------------------------------------------------------------------

## 5. 常见异常类型

``` cpp
std::out_of_range      // 越界
std::invalid_argument  // 参数非法
std::runtime_error     // 运行时错误
std::logic_error       // 逻辑错误
```

例如：

``` cpp
throw std::invalid_argument("参数不能为负数");
throw std::runtime_error("文件打开失败");
```

------------------------------------------------------------------------

## 6. e.what() 的作用

``` cpp
catch (const std::out_of_range& e)
{
    cout << e.what() << endl;
}
```

`what()` 返回异常的提示信息。

例如：

``` cpp
throw std::out_of_range("下标越界");
```

输出：

``` text
下标越界
```

------------------------------------------------------------------------

## 7. 为什么 catch 要写引用？

推荐：

``` cpp
catch (const std::out_of_range& e)
```

原因：

-   `const`：不修改异常对象。
-   `&`：避免对象拷贝，提高效率。

------------------------------------------------------------------------

## 8. 动态数组中的典型用法

``` cpp
T& at(int i)
{
    if (i < 0 || i >= size)
    {
        throw std::out_of_range("下标越界");
    }

    return p[i];
}
```

调用：

``` cpp
try
{
    cout << v.at(100) << endl;
}
catch (const std::out_of_range& e)
{
    cout << e.what() << endl;
}
```

------------------------------------------------------------------------

## 9. operator\[\] 与 at() 的区别

### operator\[\]

``` cpp
T& operator[](int i)
{
    return p[i];
}
```

特点：

-   不检查越界
-   速度快
-   越界属于未定义行为

### at()

``` cpp
T& at(int i)
{
    if (i < 0 || i >= size)
        throw std::out_of_range("下标越界");

    return p[i];
}
```

特点：

-   检查越界
-   更安全
-   多一次判断，略慢

------------------------------------------------------------------------

## 10. 总结

  关键字     作用
  ---------- ------------------------
  `throw`    抛出异常
  `try`      尝试执行可能出错的代码
  `catch`    捕获并处理异常
  `what()`   获取异常提示信息

一句话：

> **异常就是程序发现错误后，主动把错误抛出去，由调用者决定如何处理。**
