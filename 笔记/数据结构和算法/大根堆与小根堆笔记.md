# 大根堆与小根堆笔记

## 1. 堆是什么

堆是一种特殊的完全二叉树。

完全二叉树的特点：

```text
除了最后一层外，其他层都是满的；
最后一层的节点从左到右连续排列。
```

堆通常分为：

```text
大根堆（最大堆）
小根堆（最小堆）
```

堆只保证父节点和子节点之间满足大小关系，并不保证整棵树完全有序。

---

## 2. 大根堆

大根堆也叫最大堆。

规则：

```text
每个父节点都大于等于它的子节点
```

所以根节点一定是整个堆中的最大值。

例如：

```text
        90
       /  \
     70    80
    / \    /
   20 50  30
```

---

## 3. 小根堆

小根堆也叫最小堆。

规则：

```text
每个父节点都小于等于它的子节点
```

所以根节点一定是整个堆中的最小值。

例如：

```text
        10
       /  \
     20    15
    / \    /
   40 30  50
```

---

## 4. 堆不是完全有序

大根堆只保证：

```text
父节点 >= 子节点
```

小根堆只保证：

```text
父节点 <= 子节点
```

它不保证左孩子一定小于右孩子，也不保证左子树和右子树之间有整体大小关系。

---

## 5. 堆为什么常用数组保存

堆是完全二叉树，所以可以按照层序存入数组，不需要真的创建树节点。

例如：

```text
数组：[90, 70, 80, 20, 50, 30]
```

对应：

```text
        90
       /  \
     70    80
    / \    /
   20 50  30
```

---

## 6. 下标关系

假设数组下标从 `0` 开始，当前节点下标为 `i`：

```text
左孩子下标：2 * i + 1
右孩子下标：2 * i + 2
父节点下标：(i - 1) / 2
```

最后一个非叶子节点下标：

```text
n / 2 - 1
```

---

## 7. 插入元素

插入时：

1. 把新元素放到数组末尾；
2. 与父节点比较；
3. 不满足堆规则就交换；
4. 一直向上调整。

这个过程叫：

```text
向上调整
上浮
sift up
```

大根堆中新元素比父节点大时上浮。

小根堆中新元素比父节点小时上浮。

---

## 8. 删除堆顶

删除根节点时：

1. 用数组最后一个元素覆盖根节点；
2. 删除最后一个元素；
3. 从根节点开始向下调整；
4. 与合适的子节点交换。

这个过程叫：

```text
向下调整
下沉
sift down
```

大根堆中要和两个孩子中较大的那个交换。

小根堆中要和两个孩子中较小的那个交换。

---

## 9. 建堆

把普通数组调整成堆，叫建堆。

常见方法：

```text
从最后一个非叶子节点开始，
依次向前进行向下调整。
```

起始下标：

```text
n / 2 - 1
```

建堆时间复杂度：

```text
O(n)
```

---

## 10. 时间复杂度

| 操作 | 时间复杂度 |
|---|---|
| 查看堆顶 | `O(1)` |
| 插入元素 | `O(log n)` |
| 删除堆顶 | `O(log n)` |
| 建堆 | `O(n)` |
| 查找任意元素 | `O(n)` |

堆适合快速获取最大值或最小值，但不适合查找任意元素。

---

## 11. 堆排序

堆排序通常使用大根堆进行升序排序。

过程：

1. 把数组调整为大根堆；
2. 根节点是最大值；
3. 根节点与末尾元素交换；
4. 缩小堆的有效范围；
5. 对根节点继续向下调整；
6. 重复直到完成排序。

```text
升序排序：常用大根堆
降序排序：常用小根堆
```

时间复杂度：

```text
O(n log n)
```

额外空间复杂度：

```text
O(1)
```

堆排序通常是不稳定排序。

---

## 12. C++ 优先队列

默认是大根堆：

```cpp
priority_queue<int> q;
```

常用操作：

```cpp
q.push(value);
q.top();
q.pop();
q.empty();
q.size();
```

小根堆：

```cpp
priority_queue<int, vector<int>, greater<int>> q;
```

需要：

```cpp
#include <queue>
#include <vector>
#include <functional>
```

---

## 13. 大根堆和小根堆对比

| 对比项 | 大根堆 | 小根堆 |
|---|---|---|
| 父子关系 | 父节点 >= 子节点 | 父节点 <= 子节点 |
| 根节点 | 最大值 | 最小值 |
| 适合快速获取 | 最大值 | 最小值 |
| C++ 默认优先队列 | 是 | 否 |

---

## 14. 常见应用

- 快速取得最大值或最小值；
- 优先队列；
- 任务调度；
- Top K 问题；
- 合并多个有序序列；
- 堆排序。

---

## 15. Top K

求最大的 K 个数：

```text
维护一个大小为 K 的小根堆
```

因为堆顶是当前 K 个数中最小的那个。

新数字：

```text
小于等于堆顶：忽略
大于堆顶：删除堆顶，再插入新数字
```

求最小的 K 个数：

```text
维护一个大小为 K 的大根堆
```

---

## 16. 核心记忆

```text
堆是一棵完全二叉树。

大根堆：
父节点大于等于子节点，
根节点是最大值。

小根堆：
父节点小于等于子节点，
根节点是最小值。

数组下标从 0 开始：

左孩子 = 2 * i + 1
右孩子 = 2 * i + 2
父节点 = (i - 1) / 2

插入：
放到末尾，再向上调整。

删除堆顶：
末尾覆盖根节点，再向下调整。

查看堆顶 O(1)
插入和删除 O(log n)
建堆 O(n)
```
